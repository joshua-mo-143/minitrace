# minitrace
A repository showcasing a bare minimum sample of how the `tracing` library in Rust works.

## Architecture
This repository consists of a Crate workspace holding two folders intended to showcase two different crate architectures:
- A single crate that holds the entire crate with nothing split out
- A folder of crates that combine together to become the final crate(s)

## How to use
1) Clone this repo with `git clone https://www.github.com/joshua-mo-143/minitrace.git`.

That's pretty much it. Once you've cloned it, you can basically do whatever.
